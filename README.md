# Plot Figures for OSC-Genome Publication

Collection of R scripts used to generate figure panels for the publication:

**The Drosophila OSC Genome: A Resource for Studies of Transposon and piRNA Biology**

Handler et al., 2025

## Overview

This repository contains the plotting scripts and visualization tools used to create the figures presented in the OSC genome publication. The scripts process data from the OSC genome assembly and associated analyses to generate publication-quality figures.
